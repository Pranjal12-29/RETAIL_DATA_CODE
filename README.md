## Retail Product Data ELT Pipeline

This repository contains an end-to-end ELT pipeline for retail product analytics.

## Components
- Data ingestion using Python
- Data transformation using Pandas
- Data quality handling
- Warehouse-ready datasets
- SQL analytics queries

## Tech Stack
- Python
- Pandas
- PostgreSQL (simulated)
- SQL

## Note
A subset of an open-source retail dataset was used for demonstration purposes.
